# golden
All in one trading engine

## Features
- [x] Single data feed
- [x] Single strategy
- [ ] Multiple data feed
- [ ] Multiple strategy
- [ ] Customed backtest time range
- [x] Backtest UI
- [x] Single broker: IBKR
- [x] Live trading with single broker
